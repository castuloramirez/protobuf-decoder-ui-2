About
Local protobuf decoder using DynamicMessage + a pre-built descriptor set (all.desc).
Tip: If RAW fails, try GRPC (5-byte framing) or DELIMITED_VARINT (varint length prefix).
How To
Generate the descriptor set with:
protoc --include_imports --descriptor_set_out=all.desc -I path/to/protos path/to/protos/*.proto
Place it at src/main/resources/protos/all.desc or run with --desc.
